My Approach

How I Build for the Web

♿

Accessibility First

ADA/WCAG compliance is not a final-step audit. I write semantic HTML, test with screen readers, verify color contrast, and ensure keyboard navigability from the first line of code.

📱

Mobile-First, Responsive

Every project starts from a mobile viewport. I use fluid grids, relative units, and tested breakpoints to ensure a quality experience across all devices and screen sizes.

♻️

Green Digital Design

I minimize unused CSS/JS, optimize images, reduce HTTP requests, and choose lightweight dependencies — building faster pages that are also better for the environment.

🔄

Agile & Iterative

As a CSM-certified coordinator, I champion speed over perfection — ship, gather user feedback, improve. I use user stories and personas, maintain clean backlogs, and run focused sprint ceremonies.

🏛️

USWDS Principles

I apply U.S. Web Design System principles — clarity, accessibility, consistency, and user-centered design — to all digital portfolios, especially those serving the public sector.

📊

Data-Informed

I monitor Core Web Vitals, uptime, and UI/UX metrics to identify issues proactively — not reactively. Decisions are grounded in real performance data, not assumptions.
